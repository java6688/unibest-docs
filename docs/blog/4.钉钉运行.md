# unibest 如何运行 在钉钉小程序

package.json 中添加如下脚本：

```json
{
  "uni-app": {
    "scripts": {
      "mp-dingtalk": {
        "title": "钉钉小程序",
        "env": {
          "UNI_PLATFORM": "mp-alipay"
        },
        "define": {
          "MP-DINGTALK": true
        }
      }
    }
  }
}
```

scripts 中添加如下脚本：

```json
{
  "scripts": {
    "dev:mp-dingtalk": "uni -p mp-dingtalk"
  }
}
```
